Dans Word :

1) Mettre le texte sur 2 colonnes.

2) Pour mettre les titres en noir : menu Format > Style… > Titre / Titre 1 / … / Titre 5 > Modifier…
Cliquer sur le bouton ‘A’ pour changer la couleur.

3) Pour diminuer les interlignes : menu Format > Style… > Normal > Modifier… > Paragraphe…
Cocher la case “Ne pas ajouter d’espace entre les paragraphes de même style”.
